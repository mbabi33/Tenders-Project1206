  # Проект "Госзакупки Грузии"
  
  ## Описание проекта
  Этот проект предназначен для автоматизированного сбора, обработки и анализа данных по государственным закупкам в Грузии. Основная це
  — извлечение информации из различных документов (PDF, Excel, HTML), их классификация и загрузка в базу данных для дальнейшего анализ
  
  ## Структура проекта
  *   `config.py`: Файл конфигурации для настроек проекта.
  *   `organize_files.sh`: Скрипт для организации файлов.
  *   `rtnd.sh`: Основной запускающий скрипт для обработки данных.
  *   `download_*.py`: Скрипты для загрузки документов.
  *   `parse_*.py`: Скрипты для парсинга и обработки различных типов документов.
  *   `ML_DATA/`: Директория для хранения обработанных данных и баз данных.
      *   `T_45200000/`: Поддиректория для конкретных тендеров/данных.
  *   `docs/`: Дополнительная документация и заметки.
  *   `C17.py`, `C17_analysis.md`, `C17_code_reference.md`: Файлы, связанные с анализом C17.
  
  ## Установка
  ### Зависимости
  Проект написан на Python и требует установки следующих библиотек:
  *   `pandas`
  *   `PyMuPDF`
  *   `Tesseract`
  *   `scikit-learn` (предположительно, для ML части)
  *   `LangChain` (предположительно, для LLM части)
  *   `Ollama` (для LLM)
  *   Другие (будут добавлены по мере уточнения)
  
  Для установки Python зависимостей рекомендуется использовать `pip`:
  pip install pandas PyMuPDF scikit-learn langchain # и другие необходимые библиотеки

  Также убедитесь, что у вас установлен `Tesseract OCR` и настроены необходимые языковые пакеты.
  
  ### База данных
  Проект использует SQL базы данных (PostgreSQL/DuckDB/SQLite). Убедитесь, что у вас установлен и настроен необходимый драйвер и СУБД.
  
  ## Запуск
  ### Основной скрипт
  Для запуска основного процесса обработки данных используйте скрипт `rtnd.sh`:
  ./rtnd.sh

 Этот скрипт отвечает за последовательное выполнение загрузки, парсинга и организации данных.
 
 ### Альтернативные скрипты
 *   Для парсинга основных данных приложений: `python parse_app_main.py`
 *   Для организации файлов: `bash organize_files.sh`
