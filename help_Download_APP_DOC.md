# help_Download_APP_DOC.md

## Справка по скрипту `download_app_doc_files.py`

**Описание:** Скачиватель файлов документации тендеров.

---

### Параметры командной строки:

*   **`-c`**, **`--cpv`**
    *   **Тип:** `str`
    *   **Обязательный:** Да
    *   **Описание:** CPV код, используемый для организации файлов и привязки к проекту.
    *   **Пример значения:** `45200000`

*   **`-root`**, **`--root-dir`**
    *   **Тип:** `str`
    *   **Обязательный:** Нет
    *   **Описание:** Корневая директория для проектов. Если не указана, используется значение по умолчанию из `config.py`. Переопределяет `DEFAULT_ROOT_DIR`.

*   **`-batch_size`**
    *   **Тип:** `int`
    *   **Обязательный:** Нет
    *   **Описание:** Количество тендеров для обработки за один запуск.
    *   **Значение по умолчанию:** Все доступные тендеры.

*   **`--date-from`**
    *   **Тип:** `str`
    *   **Обязательный:** Нет (используется в паре с `--date-to`)
    *   **Описание:** Начальная дата для фильтрации тендеров по колонке `upload_date`.
    *   **Формат:** `ГГГГ-ММ-ДД` (например, `'2025-09-01'`)

*   **`--date-to`**
    *   **Тип:** `str`
    *   **Обязательный:** Нет (используется в паре с `--date-from`)
    *   **Описание:** Конечная дата для фильтрации тендеров по колонке `upload_date`.
    *   **Формат:** `ГГГГ-ММ-ДД` (например, `'2025-09-30'`)

---

## Примеры вызова:

### 1. Скачивание всех доступных файлов для конкретного CPV кода:

```bash
python3 download_app_doc_files.py -c 45200000
```

### 2. Скачивание файлов с фильтрацией по дате (например, за сентябрь 2025 года):

```bash
python3 download_app_doc_files.py -c 45200000 --date-from '2025-09-01' --date-to '2025-09-30'
```

### 3. Скачивание ограниченного количества тендеров (батчами):

```bash
python3 download_app_doc_files.py -c 45200000 -batch_size 100
```
