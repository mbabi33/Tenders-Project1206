# Система Классификации Документов Тендеров

## Предлагаемая система классификации:

1.  **`ЧЕРТЕЖИ_И_ПРОЕКТЫ`**
    *   **Описание:** Техническая и графическая документация: чертежи, планы, схемы, архитектурные и конструктивные проекты.
    *   **Ключевые слова (в именах файлов):** `ნახაზი` (чертеж), `პროექტი` (проект), `კონსტრუქცია` (конструкция), `gegma` (план), `პროფილი` (профиль).
    *   **Примеры из данных:** `ნახაზები.pdf`, `კონსტრუქციული_პროექტი.pdf`, `situaciuri_gegma_1.pdf`.

2.  **`СМЕТЫ_И_ТАБЛИЦЫ`**
    *   **Описание:** Финансовые документы, прайс-листы, формы-расценки (BoQ) и другие табличные данные. В основном, это файлы `.xlsx` и `.xls`.
    *   **Ключевые слова:** `ხარჯთაღრიცხვა` (смета), `ცხრილი` (таблица), `ღირებულება` (стоимость).
    *   **Примеры из данных:** `ცხრილი_N1.xlsx`, `ხარჯთაღრიცხვა_დანართი_N2.xlsx`.

3.  **`ДОГОВОРЫ`**
    *   **Описание:** Проекты договоров или финальные версии контрактов.
    *   **Ключевые слова:** `ხელშეკრულება` (договор, контракт).
    *   **Примеры из данных:** `ხელშეკრულების_პროექტი.pdf`.

4.  **`ПРОТОКОЛЫ_И_АКТЫ`**
    *   **Описание:** Официальные документы, такие как протоколы вскрытия, протоколы заседаний, акты выполненных работ.
    *   **Ключевые слова:** `ოქმი` (протокол).
    *   **Примеры из данных:** `ოქმი_N1.pdf`, `გამოცხადების_ოქმიSIGNED.pdf`.

5.  **`ПРОЧИЕ_ДОКУМЕНТЫ`**
    *   **Описание:** Общие приложения, гарантийные письма, описания опыта, графики работ, информация о персонале и другие документы, которые не вошли в более конкретные категории.
    *   **Ключевые слова:** `დანართი` (приложение), `გარანტია` (гарантия), `გამოცდილება` (опыт), `გრაფიკი` (график).
    *   **Примеры из данных:** `დანართი_N1.pdf`, `დანართი_N6_გრაფიკი.pdf`.

6.  **`НЕОПРЕДЕЛЕННЫЙ`**
    *   **Описание:** Категория для файлов, которые не удается классифицировать по имени.
